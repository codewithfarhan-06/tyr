# Farhan Khan 3D Portfolio

This is a static 3D portfolio site built with plain HTML, CSS, and JavaScript. No build step or dependencies are required.

## Run locally

### Option 1: Open directly
- Double-click `index.html` to open it in your browser.

### Option 2: Use a local server (recommended for pop-ups & 3D effects)
```bash
python -m http.server 8000
```
Then open:
```
http://localhost:8000/
```

## Project structure
- `index.html` – Main layout and content
- `styles.css` – 3D styling, glassmorphism, and animations
- `script.js` – Modal pop-ups and tilt interactions
